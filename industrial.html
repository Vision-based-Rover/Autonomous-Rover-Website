// Scalability Roadmap

PHASE 1

Short Term: Edge

Transitioning inference to TensorRT or Coral USB Accelerators to keep policy latency below 50ms on embedded hardware, freeing CPU cycles for perception tasks.

PHASE 2

Medium Term: Fleet

Scaling from a single rover to a coordinated multi-agent system using ROS 2's distributed discovery server for decentralised swarm coordination and collision-free path planning.

PHASE 3

Long Term: Sim-to-Real

Training in domain randomized simulation — injecting sensor noise, wheel slip, and dynamic lighting — to close the sim-to-real gap and enable zero-shot hardware deployment.
