Planned Component Breakdown

Compute — Raspberry Pi 4

The planned onboard brain. Will run ROS 2 Humble, the PPO inference loop at 15 Hz, LiDAR scan processing, and the Safety Guard Node simultaneously on 4 GB LPDDR4 RAM.

  • Quad-core Cortex-A72 @ 1.8 GHz
  • 4 GB LPDDR4 RAM
  • USB 3.0 for LiDAR & camera
  • GPIO for I²C IMU bus
Primary Compute Specced

Actuation — Differential Drive

Two independently driven wheels enable zero-radius turns and precise odometric tracking — critical for tight warehouse aisle navigation.

  • L298N dual H-bridge motor driver
  • PWM speed control per wheel
  • Quadrature wheel encoders (odometry)
  • 5 V / 3 A regulated power rail
PWM Control Encoder Feedback

Navigation — 360° LiDAR

A full 360° LiDAR unit provides the robot's primary spatial awareness — generating a dense point-cloud map of the surrounding warehouse environment for real-time navigation and obstacle avoidance.

  • RPLIDAR A1 / A2 — 360° planar scan, up to 12 m range
  • ~8,000 samples per revolution at 5–15 Hz scan rate
  • MPU-6050 IMU — 6-axis roll / pitch / yaw fusion
  • LiDAR + IMU + encoder fused via EKF for robust pose
360° LiDAR EKF Fusion Point Cloud

Vision — Box & Item Recognition

The Pi Camera is dedicated exclusively to identifying boxes and warehouse items on shelves and the floor. Navigation is handled entirely by the LiDAR — the camera is a pure recognition pipeline.

  • Pi Camera Module 3 — high-resolution item capture
  • Tiny-YOLO v4 — real-time box / SKU / label detection
  • OpenCV pre-processing — crop, contrast, barcode read
  • ESP32-CAM optional for secondary shelf viewing angle

Alternatives & Fallbacks

LiDAR Upgrade Path

The RPLIDAR A1 can be swapped for the RPLIDAR A3 or Hokuyo UST-10LX in higher-demand deployments — extending range to 25 m and boosting scan rate for faster corridors.

Upgrade Option

Edge Inference Upgrade

Policy inference and YOLO detection can migrate to a Coral USB Accelerator or TensorRT runtime — cutting combined latency below 20 ms and freeing the CPU for full LiDAR map processing.

Roadmap

IMU Upgrade Path

The MPU-6050 can be upgraded to a BNO055 for absolute orientation data and better heading stability — useful once real-world drift characteristics are measured against the LiDAR ground truth.
